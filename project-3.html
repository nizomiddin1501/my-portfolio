Project Overview

This system involves automation, systematic, efficient organization and management of educational processes for students and teachers. There are 3 different roles in the system: ADMIN, TEACHER, STUDENT.

A user logged in with the STUDENT role can access data in the system. A user logged in through the TEACHER role can create a new course and evaluate students, as well as build data. The ADMIN role can fully manage the system, including all capabilities.
